# 贡献指南

欢迎为这个非商用公益方向项目贡献规则核验、文档优化、脚本修复、压力测试和方法论改进。

## 基本原则

- 默认使用简体中文。
- 优先官方信源，避免用营销号、短视频、论坛经验替代证据。
- 不输出“必上”“保录”“精准概率”等伪确定性表述。
- 不提交真实考生隐私数据。
- 不把商业志愿填报机构的需求包装成公益贡献。

## 可以贡献什么

- 省份规则核验清单。
- 官方信源路径和引用格式。
- 候选表字段和 ledger 工具改进。
- README、免责声明、非商用声明和使用说明改进。
- Agent 压力测试案例。
- 数据库与概率模型路线图的非商业研究建议。

## 不接受什么

- 商业志愿填报产品集成。
- 未经授权的付费数据、内部资料或爬取数据。
- 未经回测的精确概率模型宣传。
- 真实学生资料、未脱敏截图或个人敏感信息。
- 删除或弱化非商用、免责声明、官方核验和隐私保护要求的修改。

## PR 前检查

- `PYTHONDONTWRITEBYTECODE=1 python3 -B life-leverage-college-admission/scripts/ledger_tool.py selftest`
- Ruby YAML/frontmatter 解析或等效校验。
- 无占位符、模板残留或长英文通用 prompt。
- 无 `__pycache__`、`.pyc`、临时 CSV 或学生隐私文件。
- README 中新增链接均能指向存在文件。

## 许可

提交贡献即表示你同意贡献内容按照本仓库的双许可非商用方案发布：文档/Skill 为 CC BY-NC-SA 4.0，脚本代码为 PolyForm Noncommercial 1.0.0。
