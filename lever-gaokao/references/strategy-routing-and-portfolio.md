# 策略路由与组合规则

> 使用边界：当需要把考生画像、宏观变量和候选池转换成多方案、组合收敛或最终排序时读取本文档；产出策略路由、方案包、审计规则和排序启发式；不要替代当年规则、位次和官方招生计划核验。

## 画像到策略路由

生成方案前，先根据 `persona-map.md` 判断主画像、次级约束和长期路径。候选方案必须标注画像逻辑，而不是只给冲稳保梯度。

| 主画像 | 优先策略 | 必须防范 |
| --- | --- | --- |
| 高分宽选择 | 强学科、拔尖培养、科研平台、本研衔接、国际化资源、顶尖行业入口 | 高分低配，只因热门城市或热门专业牺牲长期上限 |
| 中上分段 | 明确平台、专业、城市、升学或就业主轴，处理三角取舍 | 主轴不清，所有选项平均用力 |
| 中位窄选择 | 特殊身份、地理折价、行业特色、学风升学、可转换入口 | 随大流进入低杠杆结果 |
| 本科压线 | 公办底线、质量门槛、专业组下限、低成本和保底真实性 | 为本科身份接受高费、低质或不可接受调剂 |
| 本专科临界 | 本科执念审计，比较低质量本科、职业本科、优质高职和专升本 | 把本科身份当成唯一价值 |
| 专科/高职扩展 | 行业技能、城市产业、证书、订单班、实训和升本通道 | 只看校名，不看就业和技能壁垒 |
| 复读临界扩展 | 填报结果与复读收益对比，评估失误原因、提分空间、心理成本和政策变化 | 情绪化复读或情绪化放弃复读 |

长期路径路由：

| 长期路径 | 方案侧重 | 核心审计 |
| --- | --- | --- |
| 升学跳板 | 学风、保研/考研、导师资源、课程基础 | GPA 可维护性、升学去向、专业课难度 |
| 就业落地 | 城市产业、实习、校招、专业壁垒 | 实习可达性、就业质量、城市成本 |
| 体制考试 | 岗位适配专业、招录限制、资格条件 | 专业代码、政审体检、户籍和地区规则 |
| 证照职业 | 执业资格、实习实践、培养资质 | 准入门槛、考试周期、失败成本 |
| 行业系统 | 行业院校、系统认可、对口专业 | 是否仍需考试、资格、面试或政审 |
| 国际迁移 | 语言、预算、合作项目、海外升学 | 项目认证、费用、签证和家庭承受力 |
| 家庭资源承接 | 家庭行业、人脉、城市落脚点 | 资源是否真实可用，是否符合学生意愿 |
| 探索保留 | 大类培养、转专业、辅修、通用能力 | 转向空间、专业组下限、路径失败成本 |

## 宏观变量到策略路由

涉及 AI、产业、人口、地缘、区域战略和交叉学科时，先读取 `macro-dynamics.md`。候选方案必须标注宏观逻辑和反向风险。

| 宏观逻辑 | 适合策略 | 必须审计 |
| --- | --- | --- |
| AI 增强型 | 强基础学科、工程、法律、医学、教育、公共治理、财经风控等叠加 AI 工具 | 学生是否能掌握工具、数据和行业知识，学校课程是否可支撑 |
| AI 高替代风险型 | 泛化管理、低壁垒文案、基础财会、简单数据处理等必须补第二能力 | 数学基础、实践项目、行业壁垒、转向路径 |
| 交叉学科复合型 | 传统专业 + 数据/AI/语言/法律/金融/公共治理/产业知识 | 课程是否真实、师资是否可见、就业去向是否匹配 |
| 区域国别增长型 | 外语、区域研究、国际贸易、工程出海、教育医疗服务 | 一带一路不是口号，必须映射到语言、区域、行业和岗位 |
| 供应链重构型 | 高端制造、交通能源、物流、国际贸易、合规风控 | 行业周期、区域产业、校企资源和就业质量 |
| 人口老龄化型 | 护理、康复、医学技术、公共卫生、养老服务、健康管理 | 资格准入、工作强度、地区支付能力和学生承受力 |
| 绿色低碳型 | 电力、能源、材料、环境、碳管理、政策治理 | 是否概念过热，学校实验平台和行业资源是否真实 |
| 公共治理型 | 应急安全、数字政务、基层治理、社会工作、公共管理 | 招录规则、财政能力、岗位竞争和专业代码 |
| 国家政策导向型 | 基层公共服务、偏远地区、艰苦行业、定向培养、战略行业 | 学生本人意愿、家庭支持、服务义务、政策假设、失败退路 |

输出规则：

- 不因“AI 热”直接推荐计算机或人工智能专业。
- 不因“外语被翻译替代”直接劝退外语和区域国别方向。
- 不因长期概念好听就包装低质量学校或空心专业。
- 不因“国家需要”“公共服务”叙事承诺编制、升迁、待遇或退休结果。
- 每个高宏观杠杆选项必须给出保守、基准、上行情景。

## 候选发现到策略收敛

候选池处理顺序：

1. 官方招生计划或用户表格标准化。
2. 硬规则过滤：批次、选科、限制条件、预算和不可接受项。
3. 高召回扩展：省外、偏远高平台、行业特色、特殊身份、学风升学、可转换入口和宏观变量机会。
4. 多维打标：画像逻辑、杠杆逻辑、宏观逻辑、风险标签和信源状态。
5. 风险降级：把不可达、证据弱或下限差的选项转为观察、降级或剔除。
6. 组合收敛：按志愿数量、梯度、风险相关性和学生真实偏好生成最终组合。

不应在第 3 步就急于给最终排序。早期精准感通常来自信息不足，而不是判断可靠。

## 漏选审计

最终组合前必须反向检查：

| 审计方向 | 检查问题 | 处理 |
| --- | --- | --- |
| 省外和偏远高平台 | 是否只因为远、冷门或城市不熟就遗漏更高平台 | 补入机会雷达并核验可达性 |
| 行业特色和特殊身份 | 是否只看大学/学院名称，忽略系统资源和行业认可 | 补充主管部门、章程和就业去向核验 |
| 专业组替代 | 是否遗漏同校更安全专业组或同层次替代组 | 进入专业组调剂审计 |
| 扩展画像路径 | 是否遗漏职业本科、优质高职、专升本、复读、专项或定向 | 作为扩展方案，不与普通本科简单混排 |
| 宏观变量机会 | 是否遗漏 AI 增强、区域国别、供应链、公共治理等长期变量 | 进入三情景推演 |

漏选审计发现的选项不得直接进入最终志愿表，必须先记录来源、发现方式、待核验事项和复入条件。

## 过拟合审计

最终组合前必须检查方案是否被单一叙事牵引：

| 过拟合对象 | 典型表现 | 修正方式 |
| --- | --- | --- |
| 家长期望 | 只要本地、稳定、体面，忽略学生长期发展 | 增加学生意见和长期机会冲突提醒 |
| 学生短期偏好 | 只追热门城市、热门专业、网络声量 | 补充下行风险和可替代路径 |
| 捡漏叙事 | 把升格、合并、更名当作主要依据 | 拆分基本盘、上行假设和失败成本 |
| 宏观概念 | 用 AI、一带一路、绿色低碳包装低质量专业 | 检查课程、师资、平台和就业场景 |
| 概率工具 | 只按第三方概率排序 | 回到官方规则、位次、专业组和用户约束 |
| 本省路径 | 因熟悉感忽略全国低估机会 | 增加省外对照组 |

过拟合审计发现问题时，必须下调置信度、调整名额比例或给出替代组合。

## 组合收敛规则

正式志愿组合应避免风险高度同源：

- 不把所有冲刺都放在同一高热城市或同一热门专业。
- 不把所有稳妥都押在专业组下限不清的学校。
- 不把所有保底都放在质量弱但容易录的选项。
- 不把所有高杠杆都押在同一学校升级、同一行业周期或同一宏观趋势上。
- 不把“学生能否执行”当成默认成立，应依据学习习惯、弱项、心理承受力和家庭支持判断。

组合收敛的目标不是让每个单项都看起来最漂亮，而是让整体在录取概率、可接受下限、长期上行和失败退路之间更稳。

## 常用方案包

方案包应服务不同“决策人格”，不是把所有好选项混成一个清单。用户没有明确偏好时，至少给出 3 种互相区分的方案。

| 方案包 | 目标 | 构成重点 | 必须控制 |
| --- | --- | --- | --- |
| 保守录取优先型 | 降低滑档和不可接受录取风险 | 少量冲刺、较多稳妥、明确保底 | 专业组下行风险和保底质量 |
| 均衡价值最大化型 | 兼顾平台、专业、城市和概率 | 高质量稳妥、可接受适冲、城市或行业特色冲刺 | 成本和调剂风险 |
| 高杠杆机会型 | 寻找非对称上行空间 | 平台升级、部委/行业特色、地理折价、特殊身份、可转换入口 | 基本盘、失败成本和上行假设 |
| 地域红利抢占型 | 利用城市群、产业带和国家战略区域 | 核心城市群、区域中心、产业强市、低成本城市 | 城市收益是否能被学生实际使用 |
| 交叉学科前瞻型 | 布局未来趋势但避免追概念 | 传统专业 + 数据/AI/语言/法律/金融/公共管理 | 课程、师资、平台和就业去向 |
| 公共服务延迟回报型 | 评估基层、偏远、战略行业和公共服务的长期机会 | 公共治理、应急安全、农业医疗教育、能源交通、定向/专项路径 | 学生价值观、服务义务、政策变化和失败退路 |
| 风险分散平衡型 | 为目标不清或家庭分歧保留多条路 | 平台、城市、专业、升学、就业多类选项 | 不押单一行业、城市或上行假设 |
| 升学考研导向型 | 本科作为研究生阶段升级跳板 | 学风、升学证据、目标专业兼容、低干扰环境 | GPA 可维护性和专业课负担 |
| 考公考编或教师导向型 | 匹配体制内、教师、公共服务路径 | 师范、法学、汉语言、思政、公共管理等 | 专业代码、资格、政审、体检和招录规则 |
| 就业系统导向型 | 进入特定行业系统 | 行业认可院校、对口专业、实习证书考试路径 | 区分行业相关和就业保障 |

## 专业组与服从调剂矩阵

| 风险偏好 | 推荐专业组类型 | 调剂策略 | 核心检查 |
| --- | --- | --- | --- |
| 低风险 | 师范、公安司法、海关、明确行业组 | 可积极服从 | 组内专业是否同质、是否都可接受 |
| 中等风险 | 民航、交通、电力、财经、公共管理 | 有条件服从 | 是否仍在预期行业，是否有不可接受专业 |
| 高风险 | 综合大学特色组、冷门入口组 | 谨慎服从 | 是否只是为了进校门，失败成本是否过高 |
| 不建议 | 大杂烩专业组、不可接受专业过多 | 不建议服从 | 下限不可接受即降级或剔除 |

专业组策略：

- 冲刺档优先选择组内专业数量较多、相近度高、下限可接受的专业组。
- 关注历史招不满、大小年波动和新增专业组，但必须核验当年计划。
- 专业组内只要存在多个学生明确不能接受的专业，不应靠“调剂后再说”解决。
- 填报前必须列出组内最差可接受结果。

## 专业冷热与平台交换

- 好学校可用冷门专业换平台，但要保留转专业、辅修或考研退路。
- 一般学校优先选优势专业，否则平台和专业都弱。
- 行业学校优先保留行业相关性，不要只看专业名称是否热门。
- 学风强校适合作为升学跳板，专业应服务目标研究生方向。
- 城市强但学校弱时，要证明实习和就业资源真的能被学生获取。

## 剔除或降级规则

以下情况应剔除或降级：

- 唯一卖点是“可能升格”。
- 历史位次明显高于学生区间，且没有现实抓手；若类型本身有方法论价值，可保留为机会雷达项，但不能放入正式主推。
- 专业组内不可接受专业过多。
- 学费和生活成本突破家庭边界。
- 民办、独立学院转设、中外合作性质不清或用户不能接受。
- 办学质量弱，且没有补偿性杠杆。
- 学校层级、教学质量、学风、就业质量和升学证据均弱，只剩“容易录取”。
- 近年升本、转设、停招、校区变化或品牌价值不稳定，且没有明确补偿收益。
- 学生无法执行该路径的入学后要求。
- 方案主要迎合家长面子、热门叙事或短期情绪。

## 排序启发式

同一概率带内按以下顺序排序：

1. 学生本人对可接受结果的真实偏好。
2. 专业组下行风险。
3. 平台和长期发展杠杆。
4. 城市和家庭适配。
5. 成本。
6. 证据置信度。

平行志愿不是“顺序无所谓”。系统按顺序检索，前面的可投档结果会阻断后续志愿。
